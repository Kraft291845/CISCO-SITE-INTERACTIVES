## Интерактивные задания с сайта Cisco ##

Все интерактивные задания ищите в папках по модулю
Например интерактивное задание модуля 1.2.1.1 будет находится в каталоге 1.2.1.1

Само задание в подкаталоге interactive в каталоге модуля

Также в каталоге модуля находится архив TAR для удобного скачивания интерактивного задния
